Instructions

    Welcome to the Instructions!
    This game is a 1v1 Online Ping Pong game against a highly skilled bot(MICRO) which cannot miss, thus, you cannot win.
    But you can set Records! World Records Even!
    The game starts off slow, but increases 0.5 % each time the ball is hit by any paddle.
    The game is over if MICRO gets to 4 points. MICRO scores points from you missing the ball. 4 Strikes and you are OUT OF THERE!
    Each time you return the ball, or hit the ball with YOUR paddle, you will get a point.
    I, Andy Borhade, haven't played my game much, but I did set a record I like. Try to beat it!
